🎵 SonicSync – Music Streaming Web Application

SonicSync is a modern, responsive music streaming web application built with React.js, Redux Toolkit, **Tailwind CSS**, **Node.js**, and the **Shazam Core API**. The application enables users to discover trending songs, search tracks and artists, browse genres, and enjoy real-time music playback through a clean and intuitive interface.

## ✨ Features

* 🔍 Instant search across **1,000+ songs, artists, and albums**
* 🎧 Real-time music streaming with play, pause, next, and previous controls
* ⏱️ Live song progress bar with responsive playback updates
* 🎼 Browse trending tracks, top charts, artists, and genres
* ⚡ Fast UI powered by React and Redux Toolkit with asynchronous API handling
* 📱 Fully responsive design built using Tailwind CSS
* 🌐 Custom Node.js proxy server to reduce API rate-limiting and improve application reliability
* 🚀 Deployed on Netlify for seamless accessibility

## 🛠️ Tech Stack

* **Frontend:** React.js, JavaScript, Tailwind CSS
* **State Management:** Redux Toolkit
* **Backend:** Node.js (Proxy Server)
* **API:** Shazam Core API
* **Deployment:** Netlify

## 📈 Highlights

* Built a scalable music player capable of searching and streaming over **1,000+ tracks**.
* Achieved **sub-second UI responsiveness** for playback controls and live progress tracking.
* Reduced API rate-limiting by approximately **90%** using a custom Node.js proxy server, resulting in **~85% application uptime** during testing.
